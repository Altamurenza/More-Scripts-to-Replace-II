[ENGLISH]

This folder is only for freeroam mods that use STimeCycle.lur as its base.
You can put up to 5 mods here. Make sure to rename the file according to the following names (choose one):
- MOD_1.lur
- MOD_2.lur
- MOD_3.lur
- MOD_4.lur
- MOD_5.lur

All mods in this folder will share the same room each other. Also, the room is different from mods in GROUP_02_STC folder.
There is a chance that mods are conflicting each other because they are in the same room. Try to move one of them into GROUP_02_STC folder.


[INDONESIA]

Folder ini hanya untuk freeroam mod yang menggunakan STimeCycle.lur sebagai basis skripnya.
Kamu bisa menaruh 5 mod disini. Pastikan kamu mengubah nama filenya menjadi seperti berikut (pilih satu saja):
- MOD_1.lur
- MOD_2.lur
- MOD_3.lur
- MOD_4.lur
- MOD_5.lur

Semua mod di folder ini akan berbagi ruangan satu sama lain. Juga, ruangan mereka akan berbeda dari mod-mod yang ada di folder GROUP_02_STC.
Ada kemungkinan bahwa mod-mod akan konflik satu sama lain mengingat mereka ada di satu ruangan yang sama. Coba pindahkan salah satu dari mereka ke folder GROUP_02_STC.

[ENGLISH]

This folder is only for freeroam mods that use SLvesEff.lur or Garages.lur according to https://bully-board.com/index.php?topic=25984
For mod authors, please follow the template given in the link above! It's important to avoid function repetition.

For users, you can put up to 10 mods here by renaming any SLvesEff.lur or Garages.lur to the following names (choose one):
- MOD_1.lur
- MOD_2.lur
- MOD_3.lur
- MOD_4.lur
- MOD_5.lur
- MOD_6.lur
- MOD_7.lur
- MOD_8.lur
- MOD_9.lur
- MOD_10.lur

It does not matter if you have MOD_10.lur, but you don't have MOD_1.lur file before. It's just load order.


[INDONESIA]

Folder ini hanya untuk freeroam mod yang menggunakan SLvesEff.lur atau Garages.lur mengacu kepada https://bully-board.com/index.php?topic=25984
Kalau kamu seorang penulis mod, tolong ikuti contoh yang ada di link di atas! Penting untuk menghindari pengulangan fungsi.

Untuk pengguna mod, kamu bisa menaruh 10 mod disini dengan mengubah nama skripnya dari SLvesEff.lur atau Garages.lur menjadi seperti berikut (pilih satu saja):
- MOD_1.lur
- MOD_2.lur
- MOD_3.lur
- MOD_4.lur
- MOD_5.lur
- MOD_6.lur
- MOD_7.lur
- MOD_8.lur
- MOD_9.lur
- MOD_10.lur

Tidak masalah kalau kamu punya file MOD_10.lur tetapi tidak mempunyai file MOD_1.lur sebelumnya, itu hanya antrian untuk dimuat dalam game.
